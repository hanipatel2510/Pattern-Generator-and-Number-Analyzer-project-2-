# 🎯 Pattern Generator and Number Analyzer (Python)

## 📌 Project Overview

**Pattern Generator and Number Analyzer** is a simple Python console application that demonstrates the use of:

* Loops (`for`)
* Conditional Statements (`if-else`)
* Match-Case Statement
* User Input
* Pattern Printing
* Even/Odd Number Checking
* Sum Calculation

This project is beginner-friendly and helps understand basic Python programming concepts.

## ✨ Features

### 1. Generate a Star Pattern

* Accepts the number of rows from the user.
* Prints a right-angled triangle star (`*`) pattern.
* Validates positive input.

### 2. Analyze a Range of Numbers

* Accepts a starting and ending number.
* Identifies whether each number is **Even** or **Odd**.
* Calculates the sum of all numbers in the given range.

### 3. Exit Program

* Terminates the application safely.
